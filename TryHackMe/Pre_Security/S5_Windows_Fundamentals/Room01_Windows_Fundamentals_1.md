# 1. What File System Does Windows Use?

Modern Windows uses **NTFS** (New Technology File System).

Before NTFS, Windows used:
- FAT16 / FAT32 (File Allocation Table)
- HPFS (High Performance File System)

You still see **FAT32** on USB drives and memory cards.

### ‚≠ê Why NTFS is Better?
NTFS is a **journaling file system** ‚Üí it keeps a log, so if the computer crashes, NTFS can repair files automatically.

NTFS supports:
- Files larger than 4GB  
- File & folder permissions  
- Compression  
- Encryption (EFS)  

To check your file system:
Right-click C: drive ‚Üí Properties ‚Üí File System = NTFS.

# 2. NTFS Permissions (Very Important)

On NTFS, you can decide **who can do what** with files/folders.

### NTFS Permission Types

| Permission | What it means |
|-----------|----------------|
| Full Control | Do everything (read, write, delete) |
| Modify | Change content |
| Read & Execute | View + run programs |
| List Folder Contents | See folder items |
| Read | Only view |
| Write | Add new data |

# 3. NTFS Alternate Data Streams (ADS)

ADS = secret extra data storage inside a file.

Every file normally has one data stream ‚Üí `$DATA`.

But ADS allows:
- multiple invisible streams
- Explorer does NOT show them
- Malware can hide inside ADS
- Browser also uses ADS to mark files downloaded from Internet

You will learn how to view ADS using PowerShell later.

Example (concept only):  
A file `text.txt` can secretly have another stream like:  
`text.txt:hiddeninfo`

# 5. User Account Types

Windows has **two main account types**:

| Account Type | What it can do |
|--------------|----------------|
| Administrator | Full control over system |
| Standard User | Only normal tasks (cannot change system settings) |

You are currently using an **Administrator account**.

### Where Are User Profiles Stored?

Every user gets a profile folder in:

C:\Users\Username

Example: for user Max ‚Üí `C:\Users\Max`

Each profile contains:
- Desktop  
- Downloads  
- Documents  
- Pictures  

When a new user logs in the first time, Windows creates these folders.

# 7. UAC (User Account Control)

UAC protects Windows from malware.

Even if a user is an **Administrator**, Windows DOES NOT run everything with full power.

### How UAC Works
When something needs admin rights:

- Windows shows a **UAC popup**
- User must confirm  
- Or enter admin password (if logged in as Standard User)

### Icons with a Shield
If a program shows a **shield icon**, it needs admin permissions.

When you double-click ‚Üí UAC prompt appears.

If user doesn‚Äôt approve ‚Üí program will not run.

This prevents malware from auto-installing.

# 8. Settings vs Control Panel

Windows has two places to change system settings:

### ‚≠ê Settings  
Modern interface (introduced in Windows 8).  
Mostly used for:
- Background  
- Apps  
- Network  
- Updates  
- Devices  
- Accounts  

### ‚≠ê Control Panel  
Older interface.  
Used for advanced settings:
- Uninstall programs  
- Network adapter settings  
- User accounts  
- System settings  

Sometimes Settings sends you to Control Panel automatically.

### How to Find Anything Fast?
Use Start Menu Search.

Example: type *wallpaper* ‚Üí Windows takes you to the right place.

# 9. Task Manager (The Doctor of Windows)

Task Manager shows:

How to open:
Right-click Taskbar ‚Üí Task Manager

It first opens in **Simple View** ‚Üí click **More details** for full view.

Useful tabs:
- **Processes** ‚Üí see what‚Äôs running  
- **Performance** ‚Üí CPU, RAM, Disk, Network usage  
- **Startup** ‚Üí disable apps slowing boot  
- **Users** ‚Üí see what each user is running
